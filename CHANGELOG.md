# Changelog

All notable changes to the "Retro Neon 80s" extension will be documented in this file.

## [1.0.0] - 2025-11-22

### Added
- Initial release of Retro Neon 80s theme
- Vibrant neon color palette inspired by 80s/90s aesthetics
- CRT monitor-inspired dark backgrounds
- Comprehensive syntax highlighting for:
  - JavaScript/TypeScript
  - Python
  - HTML/CSS
  - JSON
  - Markdown
  - And many more languages
- Retro terminal colors with classic green CRT aesthetic
- Vapor wave-influenced UI elements
- High contrast for excellent readability
