## ğŸ¨ Theme Development Guidelines

### Color Palette

When modifying colors, stick to the retro neon aesthetic:

- **Hot Pink** (#FF006E) - Keywords, emphasis
- **Cyan** (#00F5FF) - Functions, operators
- **Electric Purple** (#B026FF) - Classes, variables
- **Neon Green** (#39FF14) - Strings, success
- **Orange** (#FF9E00) - Numbers, warnings

### Testing Your Changes

1. Copy the theme folder to `~/.vscode/extensions`
2. Reload VS Code
3. Test with multiple languages (JavaScript, Python, HTML, CSS, etc.)
4. Check both light and dark UI elements
5. Verify terminal colors work correctly

### Code Style

- Use **JSON formatting** with 2-space indentation
- Keep color values in **lowercase hex** format
- Add **comments** to explain non-obvious color choices
- Group related settings together

## ğŸ“ Commit Message Guidelines

Examples:
- `feat: add syntax highlighting for Rust`
- `fix: improve contrast for markdown headers`
- `style: adjust terminal background color`
